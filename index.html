Impairment, not presence.

What This Changes / What Stays the Same

What This Changes

  • Centers the standard on impairment—not metabolite presence.
  • Directs Iowa HHS to issue statewide interpretive guidance.
  • Provides a concise judicial memo for uniform practice.

What Stays the Same

  • Caseworkers still act on genuine endangerment.
  • Mandatory reporting and safety standards remain.
  • Federal scheduling isn’t altered by this initiative.

FAQ

Does this change mandatory reporting?

No. Decisions hinge on impairment and objective risk, not a positive test alone.

Is recreational cannabis involved?

No. This applies to lawful patients under Iowa’s medical program.

Will this hinder child protection?

It improves it—fewer false positives and more focus on actual danger.

How should courts apply this?

Weigh functional evidence; use toxicology as context, not conclusion.
